# SpLightプラグインとは
## 2D画像に動的に陰影を付けるUEプラグイン
- 主にドット絵で使う
- 絵の影色を制御しやすくする
- 複数画像を用意して実現する
- マテリアルベース（エンジン改造の必要無し）
